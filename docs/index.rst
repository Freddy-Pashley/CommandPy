=========
CommandPy
=========

*CommandPy* is a modern, simplistic and useful command prompt. It can be used for many purposes,
from developers to general use.

*CommandPy* was founded by Fred Pashley in 2020, and is mostly programmed in Python 3.

------------
Installation
------------

`Install CommandPy by clicking on me.
<https://github.com/Freddy-Pashley/CommandPy/releases>`_

Instructions for installation:

1. Download a release from the above link.
2. Extract the ZIP folder.
3. Open the *CommandPy.exe* file.

*Note: You must have Python 3.8 or later installed to be able to use CommandPy.*
